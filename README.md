# SimpleClickTestApp

Данное приложение создано для тестирования программ работающих с исполняемыми файлами.  

Программа является простейшей игрой, в которой кнопка "телепортируется" от курсора в произвольное место и для нее устанавливаются произвольные параметры цвета/цвета текста/размеров, а игрок должен успеть на нее нажать.
